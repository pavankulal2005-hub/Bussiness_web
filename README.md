# 🚀 Business_Web

Business_Web is a modern, futuristic, and fully responsive business website built using pure HTML, CSS, and JavaScript.  
The project features premium UI/UX design, smooth animations, glassmorphism effects, responsive layouts, and modern business sections suitable for startups, agencies, SaaS companies, portfolios, and corporate brands.
